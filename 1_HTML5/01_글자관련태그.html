글자 관련 태그

b태그 : 단순히 글자를 굵게 표시하는 태그
i태그 : 단순히 글자를 기울이는 태그
ABC(태그는 중복 적용이 가능하다.)


strong 태그 : 중요성, 심각함 등을 표현하는 강조 태그
en 태그 : 문장 일부분의 중요성을 강조할 때 사용.

mark 태그 : 형광펜 효과를 나타내는 태그
u 태그 : under line (밑줄)을 나타내는 태그
s 태그 : 취소선(잘못됨을 표시).
del 태그 : 삭제선.(없어진 텍스트)
small 태그 : 작은 글자 표시
sup 태그(윗첨자): x2
sub 태그(아랫첨자): log10
abbr 태그 : 마우스 오버/호버 시 툴팁 출력
HTTP

태그들은 해당 태그 내애서 중첩으로 사용 가능하다.
굵은글자를 사용할 수 도 있고, 기울이거나 취소선을 넣는 등
다양하게 사용할 수 있다.
